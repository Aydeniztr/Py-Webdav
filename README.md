## Installation

• Installing via github 

```console
# clone the repo
$ git clone https://github.com/Aydeniztr/Py-Webdav

# change the working directory to Py-Webdav
$ cd Py-Webdav

# install the requirements
$ python3 -m pip install -r requirements.txt
```
